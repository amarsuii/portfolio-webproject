# portfolio-webproject
A multi-page portfolio website built using HTML, CSS, JavaScript, and jQuery, featuring sections for Resume, Biodata, Projects, and Contact. Designed with a responsive layout and clean UI for academic and personal use.

Features:
•	Multi-page website (Home, Resume, Biodata, Contact)

•	Clean, modern, and responsive UI

•	Built with HTML5, CSS3, JavaScript, and jQuery

•	Easy to customize and expand

•	Suitable for Web Technology Lab / Portfolio submissions

•	Easily deployable with GitHub Pages
